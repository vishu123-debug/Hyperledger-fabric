Accessibility commitment

This demo aims to be usable with keyboard navigation and screen readers. It uses semantic HTML landmarks, clear labels, and visible focus indicators.

Features

  • Skip-to-content link.
  • Semantic structure (header/main/footer).
  • Visible focus for interactive controls.
  • Readable layout and spacing.

Limitations

  • This is a demo interface and not a production portal.
  • Some text may be technical due to evaluation evidence requirements.
